## How It Works

1. **Choose your source** — Amazon Relay, TripDash, Excel file, API, or manual entry. Select data type (trips, drivers, vehicles, fuel receipts).

2. **Map columns and validate** — System shows column headers and asks for mapping. Preview import before confirming. Automatic validation flags issues.

3. **Review and approve** — See import summary (records imported, errors, warnings). Address any issues. Confirm to load data.

4. **Data syncs automatically** — Trips appear in dashboard. Allocations sync back to source system (if enabled). No manual confirmation needed.

## Visual Overview

The import hub is your data integration command centre, designed to make connecting all your systems simple and auditable. The data connection dashboard sits at the top, showing visual tiles for each connected system (Amazon Relay, TripDash, HR system, accounting system, and others). Each tile displays a status indicator (connected/disconnected), the last sync timestamp, and the next scheduled sync time. You see at a glance which systems are actively sharing data.

When you need to add a new data source, a simple step-by-step import wizard guides you through the process. Select your connection type (load board, CSV upload, API, HR system), and the system walks you through authentication. The field mapping interface uses drag-and-drop to map source fields to system fields—load board "Delivery Location" maps to System "Drop-off Address", for example. The system remembers your mapping for future imports of the same type, so you only map once.

Before data imports, a preview screen shows the first 10 rows so you can spot errors (wrong date format, missing addresses) before they corrupt your system. If something looks wrong, reject and re-upload. Completed imports appear in a bulk import queue showing pending, in-progress, and completed imports with record counts, error counts, and details on which specific records failed (so you can fix them and retry). You can set up scheduled imports that run automatically—daily from Amazon Relay, weekly from your HR system—so data syncs without manual work. On the export side, an export dashboard lets you choose what to export (completed trips, fuel expenses, toll costs, driver hours), select your destination (Xero, QuickBooks, SAP, custom API), and schedule it (on-demand or daily). Sync status logs show every import and export operation with success/failure status and error details, creating a full audit trail for compliance audits.
