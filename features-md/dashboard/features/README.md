# FleetYes Navigator — High-Conversion Marketing Features

This directory contains benefit-driven feature pages designed for SaaS product marketing and landing page consumption.

## Structure

Each feature page follows a strict, high-conversion format:

### 1. **Key Benefits (8 points)**
- Lead with user outcomes, not technical functionality
- Each point emphasises a measurable or tangible result
- Phrased as second-person value propositions

### 2. **How It Works (4 steps)**
- Simple, sequential user experience journey
- Focus on what the user does, not what the system does internally
- Written for non-technical readers

### 3. **Core Capabilities (8 items)**
- Reframed technical features as value-backed capabilities
- Each capability includes the "why it matters" element
- Balanced, scannable format

### 4. **Use Cases (8 scenarios)**
- Real-world, outcome-focused operational scenarios
- Tied directly to business value (efficiency, cost, speed, visibility, compliance)
- Diverse industry applications

### 5. **Integration & Compatibility (8 connections)**
- Integrations presented as business enablers
- Show how each integration improves workflows
- Demonstrate platform extensibility

## Design Principles Applied

### ✨ Conversion Optimisation
- Emotional clarity (speed, control, reliability, confidence)
- Business impact highlighted (time saved, cost reduced, errors avoided)
- Scannable formatting with consistent rhythm

### 📱 Landing Page Ready
- No section dominates visually
- Clean bullets with 1–2 line descriptions max
- Designed for mobile and desktop consumption

### 🎯 User-Centric Framing
- Written for fleet managers, operations teams, and decision-makers
- Second-person language where appropriate
- Avoids jargon or explains it with user benefit

### ✅ Strict Format Compliance
- All sections maintain exact point counts
- Consistent structure across all pages
- Unified tone and voice
