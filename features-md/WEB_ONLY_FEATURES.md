# Web-Only Features: FleetYes Dashboard

Features exclusive to the **Web Dashboard** that are not currently available in the mobile app.

---

## 1. Intelligent Shift Auto-Allocation
- **Feature:** AI-powered allocation engine considers driver hours, vehicle availability, location, and compliance rules
- **Capability:** Batch-allocate dozens of trips in seconds without manual spreadsheet hunting
- **Use case:** Operations teams bulk-assign trips to optimal drivers automatically
- **Status:** Manager-side feature; no driver interaction needed

## 2. Driver Roster & Visual Scheduling
- **Feature:** Build weekly rotas with visual calendar showing working days, rest days, holidays, and unavailability
- **Capability:** Automatic conflict detection flags violations; bulk import weeks of scheduling in one go
- **Use case:** Managers plan and organize driver schedules in advance
- **Status:** Batch scheduling tool not applicable to mobile

## 3. Vehicle Maintenance & Fleet Health Tracking
- **Feature:** Track MOT, tachograph, PMI, and downtime windows with calendar view
- **Capability:** Mark vehicles unavailable and see automatic impact on scheduling
- **Use case:** Maintenance managers monitor vehicle regulatory deadlines and service windows
- **Status:** Fleet admin feature beyond driver scope

## 4. Compliance Engine with Prospective & Retrospective Checking
- **Feature:** Real-time pre-journey compliance validation + after-the-fact auditing
- **Capability:** Drivers see what's allowed before accepting a trip; managers verify entire rotas retroactively
- **Use case:** Prevent violations and create auditable records for regulatory inspections
- **Status:** Standout feature for fleet compliance audit trails

## 5. Intelligent Toll & Expense Tracking
- **Feature:** Capture tolls, parking, fines, and ad-hoc costs against specific trips
- **Capability:** Link costs to trips for accurate cost allocation and invoice adjustments
- **Use case:** Finance teams reconcile trip-level costs with invoicing
- **Status:** Batch expense processing at trip level

## 6. Inventory Management
- **Feature:** Track spare parts, equipment, and consumables
- **Capability:** Log inventory levels, set reorder thresholds, track usage
- **Use case:** Fleet managers know stock levels before ordering more
- **Status:** Logistics/operations admin feature

## 7. Import Hub & Bulk Operations
- **Feature:** Load data from external systems via API, file upload, or browser extension
- **Capability:** Connect Amazon Relay, Geotab, load boards; sync trips automatically or import via Excel
- **Capability:** Bulk create drivers, vehicles, trips; bulk delete; bulk export
- **Use case:** High-volume data integration and batch operations
- **Status:** Data admin feature not applicable to driver-facing mobile

## 8. Organisation Settings & Permissions
- **Feature:** Role-based access control and audit logs
- **Capability:** Manage users, assign permissions by role, track who changed what and when
- **Capability:** Multi-organisation support for group operators
- **Use case:** Admin teams control access and maintain audit trails
- **Status:** System administration feature

## 9. DVLA Integration & Driver Licence Validation
- **Feature:** DVLA-integrated driver licence checks and CPC tracking
- **Capability:** Track continuing professional development, verify licence validity, audit trail for regulator
- **Use case:** Compliance teams validate driver qualifications
- **Status:** Admin-level compliance verification

## 10. Advanced Telematics Integration
- **Feature:** Geotab integration for telematics and vehicle telemetry
- **Capability:** Real-time vehicle location, idle-time tracking, harsh-braking alerts
- **Use case:** Safety monitoring and fuel efficiency optimization
- **Status:** Fleet analytics feature

## 11. Marketing Pages & Feature Documentation
- **Feature:** High-conversion marketing pages for all major features
- **Capability:** Production-ready landing pages, sales collateral, and product documentation
- **Includes:** 8 key benefits, 4 how-it-works guides, 8 core capabilities, 8 use cases, 8 integration details per feature
- **Location:** `/apps/v4/content/marketing/features/`
- **Status:** 14 feature pages with INDEX.md and README.md guides

---

## By Category

**Scheduling & Roster:** Shift auto-allocation, roster building, scheduling automation

**Fleet Management:** Vehicle maintenance tracking, inventory, telematics

**Compliance & Admin:** DVLA checks, role-based access, audit logs, multi-org support

**Data Integration:** Import hub, bulk operations, external API connections

**Trip-Level Finance:** Toll and expense tracking tied to specific trips

**Marketing & Sales:** Feature landing pages, sales collateral, documentation

**Summary:** The web dashboard is a **fleet operations command center** with features for scheduling, fleet maintenance, compliance auditing, batch operations, administrative controls, and marketing enablement. **11 major features** are exclusive to managers and administrators.
