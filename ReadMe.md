# FleetYes

FleetYes offers innovative transport management software to streamline logistics operations, improve efficiency, and enhance customer satisfaction. Perfect for businesses seeking agility in a fast-paced industry.

## Key Features

- **Operations Console**: Full console to manage all of your packages and custom systems.
- **TMS (Transport Management System)**: End-to-end management for logistics operations.
- **Driver App**: Open-source driver app for iOS & Android with job management, tracking, and electronic POD.
- **Developer API**: Easy-to-use developer console with API usage metrics and event/request logs.
- **Fleet Management**: Centralized asset management with zone and service area assignment.
- **Integrations**: Extensive list of integrations including Lala Move and Shopify.
- **Operational Tools**: Fuel tracking, parking monitoring, toll expense records, and vehicle maintenance tracking.
- **Customizable & Multi-lingual**: Open-source solution adaptable to specific business needs with global language support.

## Tech Stack

- **Frontend**: HTML5, Vanilla CSS (with Bootstrap 5), JavaScript.
- **Backend/Scripts**: PHP (for contact forms and environment management).
- **Dependencies**: 
  - [PHPMailer](https://github.com/PHPMailer/PHPMailer)
  - [phpdotenv](https://github.com/vlucas/phpdotenv)
  - Various vendor libraries (AOS, GLightbox, Swiper, etc.)

## Getting Started

### Prerequisites

- A modern web browser.
- [Composer](https://getcomposer.org/) (for PHP dependencies).
- PHP 7.4+ (if running the contact form locally).

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/acsdeveloper/fleetbase-website.git
   cd fleetbase-website
   ```

2. **Install PHP dependencies:**
   ```sh
   composer install
   ```

3. **Configure environment:**
   Create a `.env` file based on the project requirements (e.g., mail server settings).

4. **Launch the website:**
   Simply open `index.html` in your browser or serve it using a local development server.
